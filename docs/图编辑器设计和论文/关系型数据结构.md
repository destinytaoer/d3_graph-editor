# 关系型数据结构

关系型图谱必须包含：代表要素的顶点、代表关系的边

## 原始数据

```json
{
  "vertexes": [
    {
      "_id": "", // 必须
      "name": "", // 顶点显示的文字
      "type": "", // 顶点类型
    }
  ],
  "edges": [
    {
      "_id": "",
      "_from": "", // 代表从哪个顶点指向哪个顶点，是顶点 _id
      "_to": "",
      "label": "", // 边显示的文字
      "type": "", // 边类型
    }
  ]
}
```

## 预处理后的数据

```json
{
  "vertexes": [
    {
      "_id": "", // 必须
      "name": "", // 顶点显示的文字
      "type": "", // 顶点类型
    }
  ],
  "edges": [
    {
      "_id": "",
      "_from": "", // 代表从哪个顶点指向哪个顶点，是顶点 _id
      "_to": "",
      "source":"", // source / target 是必须的，初始就是 _from / _to
      "target":"",
      "label": "", // 边显示的文字
      "type": "", // 边类型
    }
  ]
}
```
