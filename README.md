# 基于 D3 的图编辑器

这是 2019 年本人的本科毕业设计，5 月前会完成主要的功能，在未来仍然会进行绘图组件库的重构和不断拓展。

## 功能简介

主要分为几个部分：

- 关系图谱的展示
- 实体和它们之间的关系可新增、编辑、删除
- 可撤销、重做
- 可导入、导出

未来规划：

- 其他图谱的组件构建
